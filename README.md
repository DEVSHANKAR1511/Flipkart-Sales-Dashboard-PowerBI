# Flipkart-Sales-Dashboard-PowerBI
Interactive Power BI dashboard built on Flipkart sales data for data-driven insights.
The Flipkart Sales Dashboard is a data visualization project that analyzes sales performance, customer trends, and product insights using interactive dashboards. The objective of this project is to transform raw sales data into meaningful insights to support business decision-making.
Objectives

Analyze overall sales performance

Identify top-performing products and categories

Track revenue, profit, and order trends

Understand customer purchasing behavior

Provide insights through interactive visualizations
